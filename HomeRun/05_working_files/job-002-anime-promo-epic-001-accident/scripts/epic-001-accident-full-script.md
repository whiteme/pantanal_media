# HomeRun 动漫宣传片完整剧本

片名：Epic 001《意外》

版本：v1.0

更新时间：2026-07-06

类型：竖屏优先动漫宣传短片，兼容 16:9 横版重构

建议时长：90-120 秒

核心卖点：当阵容被突发状况打乱，HomeRun 让教练用数据重新组织球队，让替补、队长和家长都看见比赛里的真实价值。

## 创意定位

这不是一支“App 功能说明片”，而是一集热血运动番。观众先进入暑期杯训练赛的意外，再看到 HomeRun 如何把混乱变成可执行的打线策略。第一集只讲上半局，结尾把悬念留给第二集：北洲航海家的防守能否挡住城南流星的反扑。

## 延续资产

- 美术风格：HomeRun 默认球员头像的日漫运动番人物风；官网 hero 图的夜场金光、深蓝阴影、颗粒笔触和运动科技 UI 线条。
- 主队：北洲航海家。口号：把每一次进步带向下一站。
- 对手：城南流星。口号：每一颗球都要变成故事。
- 主角：林澈，7 号，CF/P，二刀流队长。
- 教练：周远航，北洲航海家总教练；沈嘉宁，打击与 AI 动作分析教练。
- 对手关键角色：石屿，城南流星先发投手；赵燃，城南流星逆风游击。

## 故事梗概

暑期杯训练赛当天，北洲航海家与城南流星约定 8:30 在球场集合。大部分队员已经到场，但球场附近高速路因交通管制临时封闭，两队各有 2-3 名主力无法准时到达。混乱让队员情绪被打乱，主教练周远航必须在比赛开始前重新排出打线。

沈嘉宁用 HomeRun 调出球员近期上垒率、推进能力、击球方向、替补训练记录和对手守备热区。她用“点球成金式”的数据棒球思路提醒周远航：现在不是寻找最像主力的人，而是找到最能完成当前任务的人。两人临场制定一套新打线，让替补球员从“临时顶上”变成“被数据看见的答案”。

上半局，替补球员接连上垒，把队友一个一个送到满垒。队长林澈根据 HomeRun 显示的城南守备站位和三垒方向反应缺口，选择强袭地滚球。球穿过内野缝隙后，外野处理失误，林澈一路冲回本垒，完成场内本垒打，带回 4 分。此时迟到队员终于抵达球场，气氛更紧张。字幕落下：下半局，是北洲航海家守住领先，还是流星追回比赛？

## 主要人物状态

林澈：队长，7 号，二刀流。开场负责稳定队友情绪，高潮用数据选择强袭地滚球，完成场内本垒打。他不是“天才开挂”，而是用 HomeRun 把选择做得更明确。

周远航：总教练。面对主力缺席先焦虑，随后接受数据建议，做出换打线决定。他代表教练的真实痛点：现场必须马上决策。

沈嘉宁：打击与 AI 动作分析教练。本集的 HomeRun 产品代言人。她不说教，而是把数据翻译成教练能执行、球员能相信的策略。

顾鸣：游击手，内野节拍器。临时进入二棒，承担推进与稳定节奏。

苏晴：左外野雷达。临时提前进入打线，靠耐心选球上垒。

纪安安：热角守门员。替补打席，用短打和跑垒制造满垒。

石屿：城南流星先发投手。快节奏压迫感强，制造本集前半段压力。

赵燃：城南流星游击。结尾给出反击眼神，连接第二集。

## 产品露出策略

1. 阵容维护：交通意外后，教练在 HomeRun 中查看出勤和默认打线。
2. 数据打线：沈嘉宁调出近期上垒率、推进打、击球方向和替补训练表现。
3. 对手分析：HomeRun 以非真实品牌 UI 的方式呈现守备热区、三垒反应和外野补位。
4. 专业计分：上半局每次上垒与推进被实时记录。
5. 成长数据：替补球员的训练记录成为临场决策依据，强调“进步留脚印”。

### 00:00-00:07 冷开场：高速封闭

画面：清晨金色阳光照进高速入口，电子路牌闪烁“临时交通管制”。车流停在匝道外，几名穿球衣的少年焦急看手机。

音效：远处汽车喇叭、手机震动、心跳式鼓点渐入。

手机群聊弹窗：

- “高速封了？”
- “我还有两个出口。”
- “开赛时间来不及了。”

旁白：有些比赛，还没开打，就先被意外投了一颗坏球。

### 00:07-00:15 球场集合：8:30

画面：球场入口大钟显示 8:30。北洲航海家和城南流星的队员陆续热身。深蓝球衣与流星红球衣在晨光里形成对比。

字幕：暑期杯训练赛 / 北洲航海家 vs 城南流星 / 集合时间 8:30

动作：林澈戴上 7 号帽子，扫视队友；另一侧石屿开始投热身球，球砸进捕手手套，声音很重。

林澈：人还没齐。

顾鸣：我们少了三棒和四棒。

### 00:15-00:25 意外扩大：两个队都缺人

画面：城南流星休息区也出现混乱。教练马驰一边接电话一边看名单，赵燃把手套摔进包里又捡起来。

马驰：我们也少两个。比赛照打，谁先稳住谁就有机会。

画面切回北洲航海家。几个替补队员站在打线板旁，眼神不确定。纪安安握紧球棒，苏晴低头看鞋钉。

周远航：默认打线不能用了。

### 00:25-00:36 HomeRun 上场：从混乱到工作台

画面：沈嘉宁打开平板上的 HomeRun。镜头从焦躁的人群切到清晰的界面：球员资料、默认打线、近期训练、上垒率、推进成功率、击球方向。

视觉要求：界面不可出现真实可操作 UI 细节或密集小字，只用清楚的模块感、数据线和高亮卡片表现。

沈嘉宁：先别找“谁像主力”。我们找“现在谁能把这局接起来”。

周远航：你想重排整条打线？

沈嘉宁：不是赌，是用证据排。

### 00:36-00:48 数据棒球决策：点球成金式打线

画面：休息区战术板与 HomeRun 数据层叠在一起。顾鸣的“推进打”、苏晴的“选球与守备范围”、纪安安的“短打训练记录”、林澈的“击球方向与跑垒贡献”被依次高亮。

沈嘉宁：那部讲数据棒球的电影里，真正重要的不是明星缺席后谁来表演，而是谁能让下一名队友前进。

周远航沉默一秒，抬头。

周远航：顾鸣二棒，苏晴提前，纪安安上场。林澈，留在关键位。

林澈：收到。

旁白：当每一次训练都被记录，替补就不只是替补。

### 00:48-01:02 上半局开始：替补把路铺出来

画面：石屿站上投手丘，快节奏投球。镜头跟随棒球穿过晨光，进入打击区。

镜头 1：苏晴选掉外角坏球，四坏上垒。HomeRun 计分界面记录：上垒。

镜头 2：顾鸣轻推右半边，跑者推进。休息区响起掌声。

镜头 3：纪安安短打，球沿三垒线滚动，城南三垒手夏禾冲上来处理，传球慢半拍。

画面：垒包灯光依次亮起，一垒、二垒、三垒全部有人。

队友喊声：满垒！

### 01:02-01:14 队长打席：看见弱点

画面：林澈走进打击区。城南内野前压，三垒方向稍空。HomeRun 的数据线在画面边缘浮现：守备站位、反应热区、地滚球通过率。

沈嘉宁低声对周远航：他们三垒前压，游击补位慢半步。林澈最近地滚球速度够。

周远航看向林澈，只做一个向下压手势。

林澈点头。

林澈内心独白：不是一定要打最远。是把球打到比赛最需要的地方。

### 01:14-01:28 强袭地滚：场内本垒打

画面：石屿投出速球。林澈挥棒，球变成一条贴地金线，强袭穿过三游间。

动作节奏：

- 三垒跑者冲回本垒，1 分。
- 二垒跑者绕过三垒，2 分。
- 一垒跑者加速，外野回传偏高，3 分。
- 林澈冲过二垒，头盔飞起一点，继续向三垒。

画面：城南外野手捡球时被金色数据线和扬尘交织包围，传球慢了一瞬。

队友喊声：回家！回家！

画面：林澈扑向本垒，手掌碰板。尘土和金光炸开。

计分牌：4 - 0

旁白：一条打线，不只靠最强的人。它靠每一个被看见的选择。

### 01:28-01:40 迟到队员抵达：紧张升级

画面：高速路口的迟到队员背着包冲进球场。北洲航海家的主力和城南流星的迟到队员几乎同时到达。

迟到队员：教练！我们到了！

画面：周远航没有立刻松一口气，而是看向计分牌和 HomeRun 上的局面记录。马驰在另一侧整理帽檐，赵燃盯着林澈。

赵燃：四分而已。下半局，换我们了。
