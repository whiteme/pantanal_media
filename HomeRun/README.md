# HomeRun 社媒素材项目

这个目录用于集中管理 HomeRun 项目的社媒素材制作。

## 目录结构

- `00_brief/`：项目需求、投放目标、平台规格、创意方向。
- `01_brand/`：品牌资料、Logo、字体、色彩、视觉规范。
- `02_source_assets/raw/`：原始素材，如照片、视频、录音、客户提供文件。
- `02_source_assets/reference/`：参考素材、竞品案例、灵感图。
- `03_copy/`：标题、正文、脚本、口播稿、标签文案。
- `04_templates/`：可复用模板，如 Canva、Figma、PSD、AE、PR 模板。
- `05_working_files/`：制作中的工程文件。
- `06_exports/`：最终导出的社媒素材。
- `07_archive/`：过期版本、历史交付物和备份。

## 命名建议

推荐文件名格式：

```text
YYYYMMDD_platform_format_topic_version.ext
```

示例：

```text
20260705_instagram_reel_launch_v01.mp4
20260705_xhs_cover_homepage_v02.png
20260705_tiktok_script_launch_v01.md
```
